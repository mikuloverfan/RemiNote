# RemiNote

Open-source handwriting notes for Obsidian.

RemiNote aims to bring a smooth, natural handwriting experience into Obsidian while keeping notes local-first, customizable, and community-driven.

## Features

* Infinite canvas / paper workflow
* Handwriting support
* Pencil and brush engine
* Fast local storage
* Open source
* Continuous updates

## Installation

### Manual

1. Download `main.js`, `manifest.json`, `styles.css`
2. Put them into:
   `.obsidian/plugins/RemiNote/`
3. Restart Obsidian
4. Enable plugin in settings
